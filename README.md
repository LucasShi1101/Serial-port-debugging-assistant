Serial Port Debugging Assistant (串口调试助手)
这是一个功能强大、界面简洁的串口调试工具，专为嵌入式工程师、IoT 开发者及硬件爱好者设计。支持常规的串口收发、多格式显示以及实时数据交互，旨在提升硬件调试效率。

🚀 主要功能
基础配置：支持自动搜索可用串口，自由配置波特率（支持自定义）、校验位、数据位和停止位。

多格式支持：

发送/接收数据支持 ASCII (String) 和 HEX (十六进制) 切换。

支持多种字符编码（如 UTF-8, GB2312 等）。

实时显示：

毫秒级时间戳记录，方便分析数据帧间隔。

自动滚屏查看最新数据。

支持收发计数重置。

便捷操作：

循环发送：可设置周期间隔，自动循环发送指令。

快捷指令：支持预设多组常用命令，一键发送。

日志保存：支持将调试日志一键导出为 .txt 或 .log 文件。

高性能：经过优化，在高波特率下连续接收不丢包、不卡顿。

📖 使用指南
连接硬件：通过 USB 转 TTL 或 RS232 连接设备到电脑。

选择端口：打开软件后点击“刷新端口”，在下拉菜单中选择对应的 COM 口。

参数设置：匹配设备的通信参数（默认通常为 9600, 8, N, 1）。

开启调试：点击“打开串口”，开始发送或接收数据。
